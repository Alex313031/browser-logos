How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/v8-ignition`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/v8-ignition`

* Use [`cdnjs`][cdnjs].
