How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/chromium`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/chromium`

* Use [`cdnjs`][cdnjs].
