How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/spidermonkey`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/spidermonkey`

* Use [`cdnjs`][cdnjs].
