## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/iridium`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/iridium`

* Use [`cdnjs`][cdnjs].
