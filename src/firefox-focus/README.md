## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox-focus`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/firefox-focus`

* Use [`cdnjs`][cdnjs].
