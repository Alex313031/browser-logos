How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/edge-beta`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/edge-beta`

* Use [`cdnjs`][cdnjs].
