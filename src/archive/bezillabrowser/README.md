How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/bezillabrowser`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/bezillabrowser`

* Use [`cdnjs`][cdnjs].
