How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/safari_8-13`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/safari_8-13`

* Use [`cdnjs`][cdnjs].
