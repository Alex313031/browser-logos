How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/chromium_1-11`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/chromium_1-11`

* Use [`cdnjs`][cdnjs].
