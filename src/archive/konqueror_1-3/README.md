How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/konqueror_1-3`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/konqueror_1-3`

* Use [`cdnjs`][cdnjs].
