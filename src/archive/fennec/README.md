How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/fennec`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/fennec`

* Use [`cdnjs`][cdnjs].
