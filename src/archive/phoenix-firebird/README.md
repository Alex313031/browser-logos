## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/phoenix-firebird`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/phoenix-firebird`

* Use [`cdnjs`][cdnjs].
