## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/chrome-canary_19-48`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/chrome-canary_19-48`

* Use [`cdnjs`][cdnjs].
