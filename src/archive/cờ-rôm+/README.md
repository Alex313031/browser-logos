## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/co-rom`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/co-rom`

* Use [`cdnjs`][cdnjs].
