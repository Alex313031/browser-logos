## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox-aurora_23-35`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/firefox-aurora_23-35`

* Use [`cdnjs`][cdnjs].
