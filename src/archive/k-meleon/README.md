How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/k-meleon`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/k-meleon`

* Use [`cdnjs`][cdnjs].
