## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/chrome-android_18-36`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/chrome-android_18-36`

* Use [`cdnjs`][cdnjs].
