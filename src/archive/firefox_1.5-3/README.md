How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox_1.5-3`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/firefox_1.5-3`

* Use [`cdnjs`][cdnjs].
