How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/aol-explorer`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/aol-explorer`

* Use [`cdnjs`][cdnjs].
