## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/iceape`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/iceape`

* Use [`cdnjs`][cdnjs].
