## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/phantomjs`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/phantomjs`

* Use [`cdnjs`][cdnjs].
