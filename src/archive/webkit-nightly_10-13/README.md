How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/webkit-nightly_10-13`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/webkit-nightly_10-13`

* Use [`cdnjs`][cdnjs].
