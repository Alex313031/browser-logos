## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/360-secure`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/360-secure`

* Use [`cdnjs`][cdnjs].
