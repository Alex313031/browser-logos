How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/webpositive`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/webpositive`

* Use [`cdnjs`][cdnjs].
