## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/srware-iron`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/srware-iron`

* Use [`cdnjs`][cdnjs].
