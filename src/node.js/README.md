How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/node.js`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/node.js`

* Use [`cdnjs`][cdnjs].
