How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/web`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/web`

* Use [`cdnjs`][cdnjs].
