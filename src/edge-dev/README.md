How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/edge-dev`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/edge-dev`

* Use [`cdnjs`][cdnjs].
