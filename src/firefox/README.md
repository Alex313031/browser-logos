How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/firefox`

* Use [`cdnjs`][cdnjs].
