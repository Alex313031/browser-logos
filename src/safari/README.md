How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/safari`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/safari`

* Use [`cdnjs`][cdnjs].
