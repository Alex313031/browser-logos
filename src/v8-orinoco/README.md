## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/v8-orinoco`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/v8-orinoco`

* Use [`cdnjs`][cdnjs].
