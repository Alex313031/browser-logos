## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/qutebrowser`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/qutebrowser`

* Use [`cdnjs`][cdnjs].
