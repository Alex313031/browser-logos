How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/midori`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/midori`

* Use [`cdnjs`][cdnjs].
