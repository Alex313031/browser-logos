How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/webkit`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/webkit`

* Use [`cdnjs`][cdnjs].
