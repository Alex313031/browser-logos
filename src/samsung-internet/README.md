How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/samsung-internet`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/samsung-internet`

* Use [`cdnjs`][cdnjs].
