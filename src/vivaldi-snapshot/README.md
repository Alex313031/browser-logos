## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/vivaldi-snapshot`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/vivaldi-snapshot`

* Use [`cdnjs`][cdnjs].
