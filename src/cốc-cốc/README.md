How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/coc-coc`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/coc-coc`

* Use [`cdnjs`][cdnjs].
