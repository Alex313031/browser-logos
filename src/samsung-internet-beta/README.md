## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/samsung-internet-beta`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/samsung-internet-beta`

* Use [`cdnjs`][cdnjs].
