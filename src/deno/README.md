How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/deno`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/deno`

* Use [`cdnjs`][cdnjs].
