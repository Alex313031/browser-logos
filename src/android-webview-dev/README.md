How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/android-webview-dev`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/android-webview-dev`

* Use [`cdnjs`][cdnjs].
