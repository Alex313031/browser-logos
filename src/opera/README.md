How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/opera`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/opera`

* Use [`cdnjs`][cdnjs].
