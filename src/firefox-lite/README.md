How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox-lite`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/firefox-lite`

* Use [`cdnjs`][cdnjs].
