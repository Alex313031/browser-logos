## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox-preview`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/firefox-preview`

* Use [`cdnjs`][cdnjs].
