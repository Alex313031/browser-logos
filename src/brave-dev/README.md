## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/brave-dev`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/brave-dev`

* Use [`cdnjs`][cdnjs].
