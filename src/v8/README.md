How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/v8`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/v8`

* Use [`cdnjs`][cdnjs].
