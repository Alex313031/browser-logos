How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox-nightly`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/firefox-nightly`

* Use [`cdnjs`][cdnjs].
