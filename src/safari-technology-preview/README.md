## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/safari-technology-preview`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/safari-technology-preview`

* Use [`cdnjs`][cdnjs].
