How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/chrome-dev`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/chrome-dev`

* Use [`cdnjs`][cdnjs].
